# 基于uni-app和Vue3的猜灯谜小程序

## 项目结构
uni-guess-riddle/
├── pages/
│   ├── index/          # 首页
│   └── game/          # 游戏页面
├── static/
│   └── images/        # 静态图片
├── store/             # Pinia状态管理
├── utils/             # 工具函数
└── uni.scss           # 全局样式

## 主要功能

### 🎮 游戏模式
1. 简单模式：8道简单题，适合新手

2. 普通模式：10道随机难度题（推荐）

3. 困难模式：12道题，包含高难度题目

### 🔄 随机题目系统
- 灯谜库包含400+条谜语，分类丰富

- 每次游戏随机抽取题目

- 保证每次游戏体验都不同


### 📊 智能评分系统
- 基础分：简单10分/中等20分/困难30分

- 时间奖励：答对越快，加分越多（最高30分）

- 提示扣分：使用提示扣除5分

### 📈 详细数据统计
- 最终得分

- 用时统计

- 答对题数

- 准确率

- 游戏模式记录

### 🎯 游戏特色
- 实时进度显示

- 美观的动画反馈

- 题目回顾功能

- 本地记录存储

- 响应式设计
